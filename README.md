# MUI GUI Demo

This project demonstrates various GUI elements using Material-UI (MUI) in React. It includes images, videos, textbox, dropdown, radio button, checkbox, tables, sliders, buttons, and icons. Navigation and responsiveness are implemented. Test scenarios for each element are provided.

## Structure
- `src/App.js`: Main app with navigation
- `src/HomePage.js`: GUI elements showcase
- `src/TestCasesPage.js`: Sample test scenarios
- `src/App.css`: Custom styles for responsiveness
- `public/index.html`: HTML template

## How to Run
1. Install dependencies:
   ```
   npm install
   ```
2. Start the development server:
   ```
   npm start
   ```

## Assets
- Images and videos use public URLs as placeholders. Replace with your own assets if needed.
